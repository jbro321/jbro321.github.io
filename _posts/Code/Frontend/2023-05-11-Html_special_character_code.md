# HTML에서 줄바꿈 없이 공백 넣는 방법

HTML에서 띄어쓰기를 하고싶은데, 띄어쓰기는 적용이 안된다면 이 방법을 사용해보세요!

```html
&nbsp;
```

- nbsp : Non-breaking Space

HTML에서는 HTML문법과 혼용되는 특수한 기호들을 표현하는 특수문자 코드가 있습니다.

띄어쓰기도 그 중에 하나이고요!

아래는 예시입니다.

```html
<a href="#test">#test</a>
&nbsp;
<a href="#test">#test</a>
```

출력값
```html
#test   #test
```
